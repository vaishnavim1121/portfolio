# Vaishnavi M — Portfolio

This repository contains a lightweight, minimal portfolio site for Vaishnavi M.

Files in this repo:
- `index.html` — main portfolio page (already added).
- `Vaishnavi_Resume_ATS.docx` — placeholder resume file (replace with your real resume to enable downloads).
- `README.md` — this file.
- `LICENSE` — MIT license.
- `.nojekyll` — prevents GitHub Pages from using Jekyll.

How to preview locally:

1. Clone the repo:
   git clone https://github.com/vaishnavim1121/portfolio.git
2. Serve locally (python):
   cd portfolio
   python3 -m http.server 8000
   open http://localhost:8000

Deploy (quick):
- Enable GitHub Pages in this repository's Settings → Pages → Source: Branch `main`, Folder `/`.
- Or connect this repo to Vercel/Netlify for automatic deploys.

Replace the placeholder resume with your real resume file and push to the repository to enable the "Resume" download link on the site.
